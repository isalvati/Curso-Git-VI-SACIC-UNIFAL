Git

Controle de versões

Este curso irá introduzir os conceitos básicos do uso de controle de versões com o Git, visto que o controle de versões é uma habilidade importante para qualquer desenvolvedor que trabalha em um projeto, principalmente se for em equipe, e deve ser utilizado de forma eficaz para o desenvolvimento fluir da melhor forma possível.
